# Гра "Вгадай слово" (Java)

## Опис
Консольна гра, в якій користувач намагається відгадати випадково вибране слово. Після кожної спроби надається підказка щодо відповідності кожної літери секретному слову.

## Вимоги

### Функціональні вимоги
1. Програма обирає випадкове слово зі словника.
2. Користувач вводить слова однакової довжини із секретним словом.
3. Після кожної спроби програма виводить підказку:
   3.1 літера на правильному місці;
   3.2 літера є у слові, але на іншому місці;
   3.3 літери немає у слові.
4. У користувача є 6 спроб.
5. Гра завершується перемогою або програшем.

### Нефункціональні вимоги
1. Консольний інтерфейс.
2. Модульна структура коду.
3. Кросплатформеність.
