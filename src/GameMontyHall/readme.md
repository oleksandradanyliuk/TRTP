# Гра "Три двері" (Monty Hall)

## Опис
Консольна гра, в якій гравець обирає одну з трьох дверей. За одними дверима знаходиться приз, за двома — порожньо. Після початкового вибору ведучий відкриває одну з порожніх дверей, після чого гравець може залишитися при своєму виборі або змінити його. Гра показує результат — виграш чи програш.

## Вимоги

### Функціональні вимоги
1. Програма випадково розміщує приз за одними з трьох дверей.
2. Гравець робить початковий вибір дверей.
3. Ведучий відкриває одну з двох інших дверей, гарантовано порожню.
4. Гравець може залишитися при своєму виборі або змінити двері.
5. Після цього програма повідомляє результат.
6. Гра повторюється за бажанням користувача.

### Нефункціональні вимоги
1. Консольний інтерфейс.
2. Модульна структура коду (логіка гри, UI, точка входу).
3. Код має бути читабельним і легко розширюваним.

## Ускладнення та розширення
1. **Випадковість ведучого** — у розширеній версії ведучий не знає, де знаходиться приз. Якщо ведучий випадково відкриває двері з призом — гра одразу завершується програшем.
2. **Різні стратегії гравця** — гравець може вибирати завжди міняти вибір або залишатися при початковому.
